# DevTasks JS

DevTasks JS é um app web criado para registrar estudos, tarefas, aulas, provas, revisões e lembretes.

O objetivo do projeto é praticar conceitos de desenvolvimento web com HTML, CSS e JavaScript, evoluindo etapa por etapa com Git, GitHub, branches, Pull Requests, Squash Merge e deploy no GitHub Pages.

## Objetivo do projeto

Criar uma aplicação simples e útil para estudantes organizarem suas atividades de estudo, com campos como matéria, tipo de registro, grau de importância, data, status e descrição.

## Funcionalidades planejadas

* Cadastrar registros de estudo
* Listar registros na tela
* Editar registros
* Excluir registros
* Marcar registros como concluídos
* Filtrar por status
* Filtrar por matéria
* Filtrar por importância
* Salvar dados no navegador com localStorage
* Exibir resumo com contadores
* Publicar o projeto no GitHub Pages

## Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
* Git
* GitHub
* GitHub Pages

## Conceitos praticados

* Estruturação de páginas com HTML
* Estilização responsiva com CSS
* Manipulação do DOM com JavaScript
